applyMoveConstraints:

- If the viewport is smaller than the game world in one of the dimensions
    - The viewport should be able to scroll 1/3rd of the way outside of the game world such that 2/3rds of the viewport is still viewing the game world in that dimension.
- If the viewport is larger than the game world in one of the dimension
    - The viewport should should be centered over the game world in the dimension that it is larger
